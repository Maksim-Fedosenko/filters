Данное приложение представляет из себя фильтр для обработки фотографий.
Рекомендуется к использованию фотографами и веб-разработчикам.
Удобство его заключается в том,что не требуется устанавливать дополнительное ПО,
достаточно иметь браузер и подключение к интернету
-------------------------------------------------------------------------------------------------------
Что умеет делать:
-Применять одиночные фильтры к фотографиям,взятым из сети интернет
-Фильтровать ссылки
-Давать подсказки
-Больше ему уметь и не надо)

Начало работы:
Для работы необходимо выбрать изображение на любом интернет-ресурсе.Затем необходимо при помощи 
ПКМ скопировать URL адрес картинки.Затем адрес вставляется в поле ввода и при помощи кнопки 
заменить данная картинка становится готовой к использованию.В ДАННЫЙ МОМЕНТ РАБОТАЕТ ТОЛЬКО 
С АДРЕСАМИ КАРТИНКИ.Для более подробных подсказок наведите курсор на интересующий вас элемент.
